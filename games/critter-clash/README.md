# Critter Clash

`Critter Clash` — это авто-баттлер про команду мемных зверей.  
Цель игрока: собрать сильный отряд, пройти как можно больше волн и набрать максимум очков.

## Основной цикл

1. Старт рана.
2. Выбор стартового состава (`3` зверя из пула).
3. Битва против вражеской команды.
4. Получение награды.
5. Переход к следующей волне и повтор цикла до поражения.

## Боевая система

- Бой идёт по ходам.
- У каждого зверя есть:
  - `HP` — здоровье,
  - `ATK` — урон,
  - `SPD` — скорость (влияет на порядок действий),
  - класс/тип питомца (внутренняя роль в балансе).
- Передняя живая единица команды считается фронтом.
- Атакующий наносит урон цели; при падении `HP` до `0` цель выбывает.
- Бой заканчивается, когда у одной из сторон не остаётся живых зверей.

## Награды и прогресс

После победы игрок получает выбор награды:

- нового питомца,
- улучшение питомца,
- предмет,
- лечение команды.

За прохождение волн и победы начисляются очки, а статистика рана фиксирует прогресс (волны, побеждённые враги, боссы).

## Интерфейс и визуальные элементы

- Карточка питомца показывает большой спрайт, имя и базовые статы (`ATK`, `SPD`, `HP`).
- В бою у врагов спрайты отражаются зеркально.
- Есть простые анимации: атака, получение урона, смерть.

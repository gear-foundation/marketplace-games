Drop optional menu art here (any size — Phaser will stretch to fit). Each
file is optional; the MenuScene falls back to a procedural version if the
image is missing.

  bg.png     — full-viewport background (wide aspect preferred, e.g. 1920×1080)
  logo.png   — game title / logo, shown near the top
  robot.png  — the red hero robot shown on the splash (512×512 works well)

When a file is present the MenuScene's preload will pick it up automatically
on page reload — no code changes needed.

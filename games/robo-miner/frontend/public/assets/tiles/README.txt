Drop the following PNG files here (48x48 ideal, any square size works — game scales them):

  dirt.png       - brown dirt block (surface row + underground fill)
  coal.png       - brown block with black speckles/veins
  iron.png       - rusty/orange-brown metallic block
  gold.png       - yellow-gold block with sparkles
  diamond.png    - cyan crystal block
  stone.png      - solid gray undrillable rock
  grass.png      - grass-topped dirt (surface row)  [optional — falls back to dirt]
  ladder.png     - wooden ladder
  pillar.png     - gray/metal support pillar

Recommended pack: Kenney "Platformer Art: Pixel Redux"
  https://kenney.nl/assets/platformer-art-pixel-redux

If a file is missing the game will fall back to the procedural Graphics
rendering for that block type — you can migrate one tile at a time.
